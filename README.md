# Laravel Design Phone Input

International phone number input with country codes for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/phone-input
```


## Usage

### Livewire Component

```blade
<livewire:ld-phone-input />
```

### Blade Component

```blade
<x-ld-phone-input />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-phone-input-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-phone-input-views
```
